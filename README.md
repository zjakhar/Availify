# Availify

Real-time inventory tracker for Walmart, Target, and Costco with watchlist and notifications

## Web UI (GitHub Pages)

Availify includes an interactive web interface hosted on GitHub Pages.

**Live site:** `https://<your-username>.github.io/Availify/`

### Features
- **Sign up / Log in / Reset password** – client-side auth stored in browser localStorage
- **Store search** – search Walmart, Target, or Costco inventory
- **Watchlist** – add, edit, and delete items to track across stores
- **Dashboard** – unified view of search results and your watchlist

### Enabling GitHub Pages
1. Go to your repository **Settings → Pages**
2. Set **Source** to `Deploy from a branch`
3. Set **Branch** to `main` (or your default branch) and folder to `/docs`
4. Click **Save** – the site will be live at `https://<your-username>.github.io/Availify/`
